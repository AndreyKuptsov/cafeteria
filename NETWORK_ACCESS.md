## 🛡️ Настройки Firewall (если не работает)

Если другие устройства не могут подключиться, возможно нужно открыть порты:

```bash
# Для Linux (если используется firewalld)
sudo firewall-cmd --add-port=3000/tcp --permanent
sudo firewall-cmd --add-port=5000/tcp --permanent
sudo firewall-cmd --reload

# Для Linux (если используется ufw)
sudo ufw allow 3000/tcp
sudo ufw allow 5000/tcp
```

## 🚀 Production Deploy (для реального использования)

Для продакшена рекомендуется:
1. Использовать Nginx как reverse proxy
2. Настроить SSL сертификат (HTTPS)
3. Использовать PM2 для управления процессами
4. Настроить домен (cafeteria.ru)

Подробнее в файле `DEPLOYMENT.md`

## 📞 Поддержка

Если возникли проблемы:
1. Проверьте, что оба сервера запущены
2. Проверьте firewall
3. Убедитесь, что устройства в одной сети
4. Проверьте логи в консоли

**Текущая конфигурация:**
- Backend: Express.js на порту 5000 (слушает 0.0.0.0)
- Frontend: Vite + React на порту 3000 (слушает 0.0.0.0)
- База данных: SQLite (локальная)
